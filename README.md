# 🌐 Full Stack Portfolio Website

Hi, I'm **Prem Kumar** 👋
This is my **full-stack personal portfolio website** built using modern web technologies.
It showcases my **skills, projects, education, and contact system** with a working backend.

---

## 🛠️ Tech Stack

### Frontend:

* React.js ⚛️
* Tailwind CSS 🎨
* Framer Motion ✨

### Backend:

* Node.js 🟢
* Express.js 🚀
* Nodemailer 📧

---

## 📂 Project Structure

```
portfolio/
├── frontend/   # React + Tailwind UI
├── backend/    # Node.js + Express API
```

---

## ✨ Features

✔ Responsive design (Mobile + Desktop)
✔ Dark / Light mode 🌙☀️
✔ Animated UI using Framer Motion
✔ Projects showcase section
✔ Skills with progress bars
✔ Education timeline
✔ Contact form (fully working)
✔ Email sending using Nodemailer 📧

---

## 📧 Contact Functionality

* User submits form
* Data sent to backend API
* Backend uses Nodemailer to send email
* Message delivered to my inbox

---


## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/premkumarmehta/prem-portfolio.git
cd portfolio
```

---

### 2️⃣ Setup Backend

```
cd backend
npm install
npm start
```

---

### 3️⃣ Setup Frontend

```
cd frontend
npm install
npm run dev
```

---

## 🚀 Deployment

* Frontend deployed on Vercel
* Backend deployed on Render
